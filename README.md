## Project Description

The Student Grade Calculator is a Python program that calculates student grades based on marks obtained in three subjects. The program collects student information, calculates averages, assigns grades, provides personalized comments, displays class statistics, allows student searching, and saves results to a file.

---

## Project Objectives

* Learn how to use conditional statements (if-elif-else)
* Practice working with lists and loops
* Implement input validation using while loops
* Handle errors using try-except
* Calculate averages and statistics
* Store and manage student records
* Save program output to a text file

---

## Features

* Calculates average marks for students
* Assigns grades automatically
* Provides personalized comments
* Validates marks between 0 and 100
* Handles invalid inputs using exception handling
* Stores student data using lists and dictionaries
* Displays results in a formatted table
* Calculates class statistics
* Searches for student records
* Saves results to a text file

---

## Technologies Used

* Python 3

## How to Run

1. Make sure Python is installed on your computer.
2. Open VS Code or Terminal.
3. Navigate to the project folder.
4. Run the following command:

python grade_calculator.py

5. Enter the required student information when prompted.
